# Etna

**Etna is a utility library inspired by RubyGem's ActiveSupport. It provides a convenient function protocol often used in Rails.**

## Installation

```elixir
def deps do
  [
    {:etna, "~> 0.1.2"}
  ]
end
```

be found at [https://hexdocs.pm/etna](https://hexdocs.pm/etna).
